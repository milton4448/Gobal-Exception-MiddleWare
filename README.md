# ASP.NET Core Global Exception Middleware

This project demonstrates how to implement Global Exception Handling in ASP.NET Core using custom middleware.

## Features
- Global exception handling
- Custom middleware
- JSON error response
- ASP.NET Core Web API

## Endpoint

GET /api/TestGlobalException

ASP.NET Core Request Logging Middleware
